### 🎯 **ПРАКТИЧЕСКОЕ ЗАДАНИЕ 1: Базовый анализ домена на Windows Server**

**Задача:** Проанализировать домен `google.com`

```powershell
# 1. Основной IP адрес (A-запись)
Resolve-DnsName google.com -Type A

# 2. IPv6 адрес (AAAA-запись)
Resolve-DnsName google.com -Type AAAA

# 3. Почтовые серверы (MX-запись)
Resolve-DnsName google.com -Type MX

# 4. Серверы имен (NS-запись)
Resolve-DnsName google.com -Type NS

# 5. Все известные записи (не ALL, но основные)
Resolve-DnsName google.com -Type ANY
```

**Ожидаемый результат:**
```
Name           Type   TTL   Section    IPAddress
----           ----   ---   -------    ---------
google.com     A      299   Answer     172.217.16.206
```

**Альтернатива через `nslookup` (для привыкших):**
```cmd
nslookup -type=A google.com
nslookup -type=MX google.com
```

### 🔎 **РАСШИРЕННЫЕ ВОЗМОЖНОСТИ**

**4. Аналог трассировки DNS запроса**
```powershell
# Показывает серверы по цепочке (упрощенный аналог trace)
Resolve-DnsName google.com -Type A -Server 8.8.8.8 -Trace
```

**5. Проверка DNSSEC**
*К сожалению, `Resolve-DnsName` не поддерживает DNSSEC напрямую. Используйте:*
```powershell
# Через DNSSEC Validator или онлайн-сервисы
# Или установите модуль: Install-Module -Name DnsServer
Get-DnsServerResourceRecord -ZoneName "google.com" -Type DNSKEY
```

**6. Рекурсивный запрос vs Authoritative**
```powershell
# Рекурсивный запрос (через ваш DNS)
Resolve-DnsName example.com

# Прямой запрос к authoritative серверу
Resolve-DnsName example.com -Server ns1.example.com
```

**Задача:** Проверить безопасность домена
```powershell
# 1. Проверить SPF запись
Resolve-DnsName example.com -Type TXT | Where-Object {$_.Strings -like "*v=spf1*"}
```

### ⚡ **ПРАКТИЧЕСКОЕ ЗАДАНИЕ 4: Мониторинг DNS изменений**

**Задача:** Написать скрипт для мониторинга изменений DNS
```powershell
# DNS_Monitor.ps1
$DOMAIN = "example.com"
$OLD_IP = (Resolve-DnsName $DOMAIN -Type A).IPAddress

function Check-DNSChanges {
    $NEW_IP = (Resolve-DnsName $DOMAIN -Type A).IPAddress
    if ($OLD_IP -ne $NEW_IP) {
        Write-Host "🚨 ВНИМАНИЕ: IP адрес изменился!" -ForegroundColor Red
        Write-Host "Старый IP: $OLD_IP"
        Write-Host "Новый IP: $NEW_IP"
        $script:OLD_IP = $NEW_IP
    } else {
        Write-Host "✅ DNS записи без изменений: $NEW_IP" -ForegroundColor Green
    }
}

# Проверять каждые 5 минут
while ($true) {
    Check-DNSChanges
    Start-Sleep -Seconds 300
}
```

### 🎓 **ПРАКТИЧЕСКОЕ ЗАДАНИЕ 5: Полный аудит домена**

**Задача:** Провести полный аудит DNS домена `your-site.com`
```powershell
# DNS_Audit.ps1
$DOMAIN = "your-site.com"

Write-Host "=== ПОЛНЫЙ DNS АУДИТ ДОМЕНА: $DOMAIN ===" -ForegroundColor Cyan

# 1. Основные записи
Write-Host "`n1. ОСНОВНЫЕ ЗАПИСИ:" -ForegroundColor Yellow
Resolve-DnsName $DOMAIN -Type A
Resolve-DnsName $DOMAIN -Type AAAA
Resolve-DnsName $DOMAIN -Type MX
Resolve-DnsName $DOMAIN -Type NS

# 2. Безопасность
Write-Host "`n2. НАСТРОЙКИ БЕЗОПАСНОСТИ:" -ForegroundColor Yellow
Resolve-DnsName $DOMAIN -Type TXT | Where-Object {$_.Strings -like "*spf1*"}
Resolve-DnsName "_dmarc.$DOMAIN" -Type TXT

# 3. Производительность
Write-Host "`n3. ПРОИЗВОДИТЕЛЬНОСТЬ:" -ForegroundColor Yellow
Measure-Command { Resolve-DnsName $DOMAIN } | Select-Object TotalSeconds

# 4. Сравнение DNS серверов
Write-Host "`n4. СРАВНЕНИЕ DNS СЕРВЕРОВ:" -ForegroundColor Yellow
$servers = @("8.8.8.8", "1.1.1.1", "9.9.9.9")
foreach ($dns in $servers) {
    $result = Resolve-DnsName $DOMAIN -Server $dns -Type A -ErrorAction SilentlyContinue
    Write-Host "Сервер $dns : $($result.IPAddress)"
}
```

### 🔍 **АНАЛИЗ ЛОГОВ WINDOWS SERVER 2022**

**Вместо Linux `/var/log/` используем:**

```powershell
# 1. Просмотр событий безопасности (аналог auth.log)
Get-WinEvent -LogName Security -MaxEvents 50 | Where-Object {$_.Id -eq 4624 -or $_.Id -eq 4625}

# 2. Поиск неудачных входов (Bruteforce)
Get-WinEvent -LogName Security | Where-Object {$_.Id -eq 4625} | Group-Object @{Expression={$_.Properties[5].Value}} | Sort-Object Count -Descending

# 3. Успешные входы
Get-WinEvent -LogName Security | Where-Object {$_.Id -eq 4624} | Select-Object -First 10

# 4. Логи DNS сервера (если роль установлена)
Get-WinEvent -LogName "DNS Server" -MaxEvents 20

# 5. Логи IIS (веб-сервер)
Get-WinEvent -LogName "Microsoft-IIS-Logging/Logs" -MaxEvents 20 -ErrorAction SilentlyContinue
```

**Поиск конкретных угроз:**
```powershell
# Подозрительные процессы (через Task Manager аналог)
Get-Process | Where-Object {$_.Path -like "*temp*" -or $_.Path -like "*tmp*"}

# Сетевые подключения (аналог netstat)
Get-NetTCPConnection | Where-Object {$_.State -eq "Established"} | Select-Object LocalAddress, RemoteAddress, OwningProcess

# Поиск в логах по ключевым словам
Get-WinEvent -LogName Security | Where-Object {$_.Message -like "*cmd.exe*" -or $_.Message -like "*powershell*"}
```
